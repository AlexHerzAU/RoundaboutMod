The Roundabout mod is meant to be a Jojo mod for Minecraft that fits well into the core gameplay loop, 
and attempts to imagine how abilities from the franchise would interact with the world of the game.

Permission is granted to create addons or contribute freely to the project. The official release will
only accept contributions which fit the intended vision of the mod. Check the license for details
(it's pretty unrestrictive)

We are not liable for any problems this mod may cause.

But if you have any problems feel free to report them, and I'll do my best to fix bugs and other issues. :)
Check the wiki in the top row of this github to learn how to use the mod.

The primary code for stand users lies in the StandUserEntity mixin for living entities, which has standPowers attached. To add a stand, we basically make a standpowers class, override any standpowers templates we want to use, and attach the standpower to a standdisc item. The stand is stored in the player as a standdisc item, and generates the stand from the attached standdisc's standpower, that's where all of the functions lie.
